== Description ==
The EdiUser plugin for WordPress is a powerful tool that allows you to manage user roles and permissions on your WordPress website with ease. You can customize and modify user roles, grant or restrict access to various features and functionalities, and ensure the right users have the appropriate privileges.


Plugins officially supported:
* WooCommerce

== Installation ==

1. Upload the plugin package to the plugins directory.
2. Activate the plugin.
3. Done! Your plugin is activated :)

== Features ==
Create, edit, and delete user roles.
Customize role names and capabilities.
Assign or remove specific capabilities to user roles.
Easily manage user permissions.
Revoke or grant access to various WordPress features.
Improve your website's security by fine-tuning user privileges.


== Usage ==
Once the plugin is activated, you can access the EdiUser from the WordPress admin panel. Here's how to use the plugin:

In your WordPress admin panel, go to "EdiUser" ."
You'll see a list of existing roles. You can customize these roles or create new ones.
To edit a role, click on it and modify the capabilities as needed.
To create a new role, use the "New Role" option.
Save your changes.


== Frequently Asked Questions ==
How do I restrict access to specific pages or content for certain user roles?
You can restrict access to content by using the "Capabilities" section in the Edit user role. Simply remove the capabilities that allow access to certain features or content from a specific user role.

Can I create custom user roles with specific permissions?
Yes, you can create custom user roles and assign specific capabilities to them using the EdiUser.
